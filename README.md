# Text-Editor
## Overview
A Text Editor that I created to learn more about Java Swing. 
This project demonstrates my growth as a developer and my understanding of Java swing.

## Features
- Open, edit, and save text files (`.txt`) 
- Close file with unsaved changes warning
- Apply Font Formatting and color selection
- Adjustable zoom
- Keyboard shortcuts
- Toggle Line numbers, status bar, and current line highlight
- Search text within the document

## Installation / Running

### Prerequisite
* Java 17 or later installed on your system

### Option 1: Double-Click
1. Download the TextEditor.jar file.
2. Simply double-click the TextEditor.jar file.
- This will launch the application if Java is properly installed.

### Option 2: Command Line
1. Download the TextEditor.jar file.
2. Open a terminal or command prompt and navigate to the folder containing the JAR file.
3. Run the program with: `java -jar TextEditor.jar`.
- If Option 1 doesn't work, use this method instead.
